# Unsupervised-ML

 Notebooks for unsupervised machine learning techniques and data analysis.

## Contents

- Clustering algorithms
- Dimensionality reduction
- Data visualization
- Model evaluation and comparison
- Experimental notebooks and reports

## Structure

- `notebooks/` — Jupyter notebooks with experiments
- `data/` — datasets used for training and testing
- `results/` — generated outputs and plots
- `report/` — project documentation and analysis

## Usage

Run the notebooks locally with Jupyter:

```bash
jupyter notebook
